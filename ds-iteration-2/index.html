How It Works

  • Select party level (1-10), size (1-6), and desired encounter difficulty.
  • Choose a monster theme or "Any" for a random mix.
  • The generator uses XP thresholds (DMG rules) to create balanced encounters.
  • Monsters are selected based on theme and XP calculations.
  • Click "Regenerate" to create a new encounter with the same parameters.
