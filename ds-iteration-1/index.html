How It Works

  • Select your party's level (1-10) and size (1-6 adventurers).
  • Choose a monster theme or "Any" for a random mix.
  • The generator will fetch monsters from Open5e API and create a plausible encounter.
  • Monsters are selected based on party level and chosen theme.
  • Click "Regenerate" to create a new encounter with the same parameters.
